# Bot Builder SDK v4 で Q&A アプリケーション (QnA Maker)

このハンズオンは、Bot Builder SDK v4 と QnA Maker を使った Q&A アプリケーションの作成手順を理解します。

ハンズオンは以下の手順で進めます。

1. Microsoft Azure のアカウントを準備する
2. 開発環境を準備する
3. QnA Maker のナレッジベースを作成する
4. Bot アプリケーションを作成する
5. Emulator で動作を確認する
